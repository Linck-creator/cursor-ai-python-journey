# Desktop Notepad

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue.svg)](https://www.python.org/)

A modern desktop notepad application built with Tkinter, featuring a clean object-oriented structure. This project demonstrates efficient GUI layout, file operations, Light/Dark mode, and best practices for responsive user interface design in Python. The application is suitable for learners interested in GUI programming, event-driven development, and user-centric application design.

---

## ✨ Features

- Create new text documents
- Open existing text files
- Save files
- Modern toolbar
- Menu bar (File and View)
- Scrollbar for text area
- Light and Dark mode (toggle)
- Unsaved changes detection
- Responsive window layout

---

## 🛠 Technologies Used

- Python 3.10+
- Tkinter (Standard Library)

## 🚀 Installation

1. Clone the repository.

   ```bash
   git clone https://github.com/Linck-creator/cursor-ai-python-journey.git
   cd cursor-ai-python-journey/desktop_notepad
   ```

2. (Optional) Create and activate a virtual environment:

<details>
  <summary>Windows (PowerShell)</summary>

  ```bash
  python -m venv venv
  .\venv\Scripts\Activate.ps1
  ```

</details>

<details>
  <summary>Unix / macOS</summary>

  ```bash
  python -m venv venv
  source venv/bin/activate
  ```

</details>

> This project uses only the Python Standard Library. No external packages are required at runtime.

## ▶️ Usage

Run:

```bash
python notes.py
```

The application will launch in a window. Use the menu or toolbar to create, open, or save notes.
