# Prime Numbers

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue.svg)](https://www.python.org/)

A Python educational project demonstrating an optimized algorithm for prime number verification using the square root method. This script demonstrates best practices in primality testing with a focus on performance and readability, making it ideal for learners exploring mathematical algorithms and algorithmic thinking in Python.

---

## ✨ Features

- Determine whether a number is prime
- Use square root optimization with `math.isqrt()`
- Skip unnecessary even-number checks
- Demonstrate efficient conditional logic
- Display results in the terminal

---

## 🛠 Technologies Used

- Python 3.10+
- Python Standard Library (`math`)

## 🚀 Installation

1. Clone the repository.

   ```bash
   git clone https://github.com/Linck-creator/cursor-ai-python-journey.git
   cd cursor-ai-python-journey/prime_numbers
   ```

2. (Optional) Create and activate a virtual environment:

<details>
  <summary>Windows (PowerShell)</summary>

  ```bash
  python -m venv venv
  .\venv\Scripts\Activate.ps1
  ```

</details>

<details>
  <summary>Unix / macOS</summary>

  ```bash
  python -m venv venv
  source venv/bin/activate
  ```

</details>

3. Install project dependencies.

   ```bash
   pip install -r requirements.txt
   ```

> This project uses only the Python Standard Library. No external packages are required at runtime.

---

## ▶️ Usage

Run:

```bash
python prime_numbers.py
```

The script executes predefined examples and prints whether each tested number is prime in the terminal.
